# InvHunter

## Setup

Setting environment of python3.8.
```
pip install -r requirements.txt
```

## Usage

### Preparation

Setting config in main.py
```
dapp = "qubit"
path = "../invFuzz/hunter/dapps" # the path of txs
sourcePath = f"dapps" # the path of source code of contract
```

### Cmd
```
python main.py
```
